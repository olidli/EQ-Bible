情商宝典小程序将您整理的 **356项知识库** 完整移植到微信生态，提供：

- 🔍 **智能搜索** — 关键词/标签检索全部内容

**核心优势**：零服务器成本 · 零维护负担 · 个人可注册

## 🚀 部署步骤（5步，约30分钟）

### 第1步：注册小程序
1. 访问 [微信公众平台](https://mp.weixin.qq.com/) → 注册 → 选择「小程序」
2. 个人可注册，**无需企业资质**
3. 设置小程序名称（建议：情商宝典）
4. 获取 AppID（开发设置中）

### 第2步：开通云开发
1. 登录小程序管理后台
2. 进入「开发」→「开发工具」→ 下载微信开发者工具
3. 打开开发者工具，导入 `miniprogram/` 目录
4. 在工具栏点击「云开发」按钮，开通云开发环境
5. 创建免费环境（个人使用足够）

### 第3步：创建数据库集合
在云开发控制台 → 数据库 → 创建集合：

| 集合名 | 权限设置 | 说明 |
|--------|---------|------|
| `knowledge_base` | 「所有用户可读」 | 356项知识库内容 |
| `user_progress` | 「每个用户仅可读写自己的记录」 | 用户学习进度 |

### 第4步：导入知识库数据
**方式A（推荐）：JSON导入**
1. 运行 `scripts/import_data.js` 生成 `export/knowledge_base.json`
2. 在云开发控制台 → `knowledge_base` 集合 → 导入
3. 选择JSON格式，自动识别字段

**方式B：云函数导入**
1. 部署 `cloudfunctions/EQMaster` 云函数
2. 调用云函数批量写入数据

### 第5步：上传并发布
1. 微信开发者工具 → 上传
2. 登录小程序后台 → 版本管理 → 提交审核
3. 审核通过后发布（约1-3个工作日）

## 🔧 本地开发

```bash
# 1. 克隆项目
git clone <repo>
cd miniprogram

# 2. 安装依赖（仅云函数）
cd cloudfunctions/EQMaster
npm install

# 3. 用微信开发者工具打开 miniprogram 目录
# 4. 开启云开发 → 上传云函数

# 5. 数据导入
node scripts/import_data.js
# 然后在云开发控制台导入生成的 JSON 文件
```
